# Configurando o Ambiente

Partindo da ideia que não tenho nenhuma experiência e não sei absolutamente nada de C, quero aprender do nivel -2 até o HPC, ou seja, irei usar LLMs para encurtar minha curva de aprendizado.

A unica coisa que hj, na inflexão do uso de inteligência artificial, acaba que a diferença entre algo prático está mais voltada a como ele abstrai e modelo a pergunta "o que quero"  que "qual linguagem usarr"

A matemática sempre foi a linguagem universal, e é a camada entre nós e qualquer sistema lógico/eletrônico, e a unica diferença entre um e outro é forma como proceça a entropia, então começo sempre pelo começo que é configurando um Seed pro meu LLM diário [(como descrito nesse artigo)](https://open.gitbook.com/~site/site_QvHjh/~/revisions/K9EFBoVzDjINqeez5ufF/llms/comecando-um-novo-projeto).

Após essa configuração inicial, começo definindo meu ambiente de produção...
