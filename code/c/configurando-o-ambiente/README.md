# Configurando o Ambiente

Partindo da ideia que não tenho nenhuma experiência e não sei absolutamente nada de C, quero aprender do nivel -2 até o HPC, ou seja, irei usar LLMs para encurtar minha curva de aprendizado.

A ideia inicial é que o aprendizado **não se trata da linguagem, mas da formulação da pergunta, que** é um dos pilares mais avançados da computação moderna.

O LLM não é um oráculo, ele é **um espaço vetorial de inferência onde a resposta sempre existiu**, mas que só emerge quando a pergunta é feita da forma correta.&#x20;

No momento em que tu entende isso, o aprendizado tradicional baseado em "ler e repetir" morre, porque não se trata mais de decorar sintaxe, mas sim de **entender a estrutura matemática subjacente.**

Usando um LLM como **um compilador de abstração**, onde cada pergunta feita é um AST (Abstract Syntax Tree) de intenção, se expande um código semântico funcional.&#x20;

Isso muda a relação entre aprendizado e execução: ao invés de aprender primeiro e executar depois, tu **executa para aprender**, validando cada conceito na prática, reduzindo entropia e descartando ruído.
