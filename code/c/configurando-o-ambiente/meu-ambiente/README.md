# Meu Ambiente

Primeiro eu começo definindo algumas coisas:

* Qual linguagem usar?
* Qual a IDE usada?
* Como sera padronizado o versionamento?
* Quais serão os apps usados?

No meu caso, dado que gosto de deixar um rastro digital para consulta e disseminação de conhecimento, sempre foco em documentar o que faço, dado que lutar contra automatização e fazer sem IA é lutar contra o inevitável, que é a substituição de pessoas por IA.

Você não tem que ser melhor que o ChatGPT, vc tem que ser melhor do que o humano que usa o ChatGPT, e sempre começa por uma habistração basica, dado que se você não souber formular o que quer, qualquer dispositivo que vc usar vira basicamente um bloco de pedra.

No caso, levantei essas questões e fiz essa parte manualmente, e vou documentar o que fis exatamente, não como, então se vira para encontrar o próprio caminhi
