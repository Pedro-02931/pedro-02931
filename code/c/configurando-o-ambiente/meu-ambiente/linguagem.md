# Linguagem

Decidi usar C, porém ao invés de importar bibliotecas nativas, tento formular sempre o conceito lógico/matemático por trás, então acaba reduzindo a possibilidade de erro por conta de bibliotecas estáticas ou dinâmicas.

Além disso, você garnte uma quantização de como os dados são processados, reduzindo a necessidade de usar bibliotecas que você nem sabe como funciona, apenas importa uma sugestão aleatória de um artigo no Reddit.

No caso, com o uso de modelos matemáticos, vc abstrai o que é processado e como é processado, e dado que o uso de IAs generativas reduz o tempo de produção de SNIPS (SNIPS, não PROJETOS), acaba sendo mais eficiente vc usar um LLM como uma biblioteca universal do que simplesmente importar.

Dessa forma, você é obrigado a usar o máximo da sua maquina, expremendo e abstraindo, tanto seu desktop quanto o seu cerebro, pq constantemente estára revendo a LÓGICA, não o PROGRAMA pronto, o que garante um sistema resiliente e adaptativo, onde você aprender a como aprender.

No caso, eu basicamente usei o conceitos para tensoriar o projeto de defini o meu nivel, tipo, não tenho experiencia, mas quero começar um projeto avançado derivado de um livro e aprender a aplicar tecnicas de HPC nele, e no final, estariei batendo de frente com uma equipe inteira do CERN, mas precisando de dois baseados e masoquismo brutal.
